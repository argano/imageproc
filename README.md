# imageproc

Image processing utilities for Node.js

## Supported Operations

- getImageInfo
- resizeAspectFit
- resizeCrop
- convertFormat

## Serverless support

- [@imageproc/cloud-functions](./packages/cloud-functions/)
  - You can easily build GCP Cloud Functions for image processing.

## Development

### Build

```
yarn build
```

### Test

```
yarn test
```
